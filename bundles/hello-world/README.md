# Hello World

A minimal starter template for building Paprwork mini-apps.

## What It Does

Displays a simple welcome page with a counter button demonstrating basic interactivity and the Liquid Glass design system.

## Use As a Template

This bundle is designed as a starting point. After importing, modify the files to build your own app:

- `index.html` - App entry point and structure
- `app.ts` - Application logic (TypeScript, auto-transpiled by Paprwork)
- `style.css` - Custom styles (Liquid Glass CSS variables available)
